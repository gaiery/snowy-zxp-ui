# snowy-zxp-ui

### 文档库地址
https://gaiery.github.io/snowy-zxp-ui/

### 快速开始

#### 1、安装组件库

```bash
npm install snowy-zxp-ui -S
```

#### 2、引用组件库

```javascript
//全部引入
import 'snowy-zxp-ui/dist/css/index.css';
import SUI from 'snowy-zxp-ui'
Vue.use(SUI)

//按需引入
import 'snowy-zxp-ui/dist/css/card.css'
import { Card } from 'snowy-zxp-ui'
Vue.use(Card)

```
